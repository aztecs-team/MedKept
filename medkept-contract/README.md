# BlockHealth

Secure decentralized electronic health records sharing system based on blockchains
Sample Contract deployed at Rinkeby test network : 0xdab005E9E9E6b36a9bBe2FC4bC7DF58c5C55f8fb


Run the code below to deploy the new contract

```shell
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network rinkeby
```
